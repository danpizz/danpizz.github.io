Una lista di articoli interessanti sull’argomento delle applicazioni multipiattaforma (native, html5 + css, PhoneGap, etc.).

Shell Apps and Silver Bullets — Post che descrive alcuni dei rischi in cui si incappa quando si cerca di risparmiare creando una app ibrida per le piattaforme mobile. Questo mio articolo ne traduce una parte.

Native, Web, or hybrid: How to choose your mobile development path — Un altro articolo, più recente (ottobre 2012), sulle opzioni disponibili per chi voglia avventurarsi nel mondo dello sviluppo multipiattaforma.
